# nest.js + typeORM monorepo

프로젝트를 진행하다보면 여러개의 서비스를 만들어야 할 때, 코드를 재사용하고 관리하는 부분은 매우 중요합니다. <br />
이런 문제를 해결하기 위한 하나의 방법으로 **모노레포** 구조를 많이 사용합니다.

모노레포 구조는 여러개의 프로젝트를 하나의 저장소에서 관리하는 방법으로, 여러 프로젝트의 코드를 한 저장소에 모아두고 공통된 코드를 공유하며 코드관리와 협업 시 효율적인 개발이 가능합니다. <br />
이번 글에서는 **nest.js**와 **typeORM**을 사용하여 모노레포 구조를 만들어보겠습니다.

## 0. Nest.js CLI 설치

모노레포를 구성하기 전에 먼저 **Nest.js CLI**를 설치해야 합니다. <br />
Nest.js CLI는 Nest.js 프로젝트를 생성하고 관리하는데 도움을 주는 도구입니다.

```bash
$ npm i -g @nestjs/cli
```
